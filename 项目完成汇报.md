## 🔧 技术架构亮点

### 现代化技术栈
```
前端: React 18.3 + TypeScript + Vite 6.0
UI: TailwindCSS + shadcn/ui + Radix UI  
后端: Supabase (企业级BaaS)
状态: Zustand + React Query
图表: Recharts
```

### 学校集成方案
```
域名: party.imust.edu.cn
认证: CAS统一认证 (cas.imust.edu.cn)
权限: 基于学校角色的权限映射
数据: 实时 + 定时 + 按需三种同步方式
```

### 安全防护体系
```
传输: HTTPS (TLS 1.3) 全站加密
存储: 数据库字段加密
访问: RBAC + RLS双重验证
监控: 实时监控 + 自动告警
审计: 完整操作日志记录
```
