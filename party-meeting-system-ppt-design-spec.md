# 党组织生活会议管理系统技术栈PPT设计规范

## 1. 设计方向与理念

### 风格精髓：Swiss Design（国际主义排版风格）

采用瑞士设计的严谨性与数学精确性，结合党政主题的权威感，打造专业技术展示风格。核心理念：客观性（设计作为中性信息通道）、清晰性（直接沟通无歧义）、功能性（形式追随功能）、数学精确性（网格化布局）。

**选择理由**：

- **严谨专业**：瑞士设计的系统化网格完美匹配技术架构展示需求，数学精确性体现技术严谨度
- **权威契合**：纯黑白基础色搭配党政红、中国金，既保持Swiss Design的克制美学，又体现党政主题的庄重感
- **功能优先**：去装饰化设计使技术信息成为焦点，架构图、技术栈列表得以清晰呈现

**核心特征**：

- 6列模块化网格系统（240px列宽，40px间距）
- Helvetica/Inter字体家族，左对齐排版为主
- 党政红(#C8102E)占比10-15%，金色(#D4A017)用于关键强调
- 禁止渐变、装饰性元素，纯几何形状
- 40-50%留白比例（技术展示需中等信息密度）

**参考案例**：IBM技术文档、MIT媒体实验室展示、瑞士航空品牌体系、国际学术会议PPT

## 3. 视觉指南

### 架构图规范

**组件表示**：
- 矩形框（1-2px黑色边框，圆角4px）
- 尺寸：宽度600-800px，高度48-80px（按内容调整）
- 标签：24px Medium，居中或左对齐（框内padding 16px）
- 层级间距：32-48px垂直间距

**连接关系**：
- 箭头：2px实线，三角箭头8×8px
- 颜色：黑色默认，党政红标注关键路径，金色数据流向
- 标注：16px文字说明关系类型（位于箭头旁12px）

**图例说明**（如需要）：
- 位置：图表右下角或底部
- 格式：符号（24×24px）+ 说明文字（18px Regular）
- 间距：图例项水平间距40px

### 技术图标规范

**来源**：仅使用SVG图标，推荐库：
- Lucide（通用图标）
- Simple Icons（品牌LOGO，保持原色）
- Heroicons（功能图标）

**尺寸规格**：
- 大图标：48-64px（封面、主视觉区）
- 中图标：32-40px（内容列表）
- 小图标：24px（装饰、辅助说明）

**样式要求**：
- 描边宽度：2-3px统一
- 颜色：单色（黑/党政红/金色）或品牌原色（React/Vue/PostgreSQL等）
- 对齐：居中对齐于所属区域
- ❌ 禁止：emoji、渐变图标、3D图标、装饰性图案

### 数据可视化

**技术栈列表**：
- 圆点标记（8px直径，党政红填充）+ 文字24px
- 缩进：32px左缩进
- 行距：32px垂直间距
- 最多7项可见（超出分页）

**对比表格**：
- 表头：28px Bold，底部边框2px党政红
- 数据行：24px Regular，行高48px
- 斑马纹：可选（浅灰#F9F9F9交替）
- 边框：仅水平线1px #CCCCCC，无垂直线
- 对齐：文字左对齐，数字右对齐

**进度指示**（如技术成熟度）**：
- 横条：宽度200-300px，高度8-12px
- 填充：党政红表示进度，浅灰#E5E5E5表示背景
- 标签：进度值64px Bold于条形右侧，单位18px

### 背景元素

**装饰线条**：
- 水平线：1-8px高度，200-400px宽度
- 位置：标题下方12-24px，或作为分隔线
- 颜色：党政红（强调）、黑色（分隔）、金色（点缀）

**网格参考线**（可选，仅设计阶段可见）**：
- 6列网格，240px列宽，40px间距
- 浅灰虚线#E5E5E5，1px宽度
- 生产环境隐藏

**几何形状**：
- 类型：矩形、圆形、直线（禁止有机曲线）
- 用途：标注、强调、分隔
- 样式：1-2px描边，透明或纯色填充
- 最多3个形状/幻灯片

### Master元素（全局一致）

**页码**：
- 位置：右下角（距右边距120px，距底边距40px）
- 格式：当前页/总页数（如"5 / 10"）
- 样式：16px Regular，#666666
- 分隔符：" / "（空格包围）

**Logo位置**：
- 党徽：右上角（距右边距120px，距顶边距40px）
- 尺寸：40×40px（非封面页）
- 颜色：金色#D4A017单色版本

**页脚文字**（可选）**：
- 位置：底部居中（距底边距24px）
- 内容结构：机构名称（如"XX大学党委组织部"）
- 样式：14px Light，#999999

### 图片使用指南（供实现agent参考）

**架构图/流程图**：使用在线绘图工具生成（如Excalidraw、Draw.io），导出SVG格式，保持黑白+红金配色

**技术Logo**：从Simple Icons、各技术官网获取官方SVG LOGO，保持品牌原色

**装饰图形**：使用几何形状SVG（矩形、圆形、线条），避免复杂插图

**禁止**：库存照片、抽象背景图、纹理、人物图片（除非为团队介绍需求，此PPT不涉及）

### 动画效果

**幻灯片切换**：
- 首选：无过渡（即时切换）
- 备选：简单淡入淡出（0.3s，ease-out）
- 禁止：滑动、缩放、旋转、3D翻转

**元素入场**：
- 标题：淡入（0.3s，延迟0s）
- 内容块：从左滑入（0.4s，延迟0.2s，ease-out）
- 列表项：依次淡入（0.3s，每项延迟+0.1s，stagger效果）
- 图表：整体淡入（0.5s，延迟0.3s）

**交互效果**（可选）**：
- 悬停：卡片提升2px + 阴影（0 4px 12px rgba(0,0,0,0.1)），过渡0.2s
- 点击：无效果（静态展示PPT）

**可访问性**：
- 支持prefers-reduced-motion，禁用所有动画
- 动画仅为视觉增强，不影响信息传达

## 4. 实现限制

**MANDATORY（强制要求）**：

❌ **禁止emoji**：严禁在任何位置使用emoji（🎨📊✅等），使用SVG图标替代（Lucide/Heroicons/Simple Icons）

❌ **禁止具体内容**：模板中不得出现真实公司名、项目名、联系方式、具体数据、日期、金额、文件路径

✅ **仅视觉模式**：描述布局结构、排版样式、配色方案、尺寸规格，使用占位符表示（如"{{title}}"、"系统名称"、"联系信息块3行结构"）

✅ **SVG图标**：仅使用专业图标库（Lucide/Heroicons/Phosphor/Simple Icons），尺寸24-64px，2-3px描边宽度

✅ **Token化**：所有数值（字号、间距、颜色）必须来自design-tokens.json，不使用任意值

✅ **网格对齐**：所有元素必须对齐6列网格系统（列宽240px，间距40px，边距120px/80px）

✅ **无渐变**：Swiss Design严格禁止渐变（线性、径向、圆锥），仅使用纯色

✅ **对比度**：文字对比度≥7:1（WCAG AA），关键元素≥10:1

✅ **信息密度**：每页≤7行等效内容（22-24px字号），超出使用分页策略（Part 1/2）

**实现agent须知**：
- 根据实际技术栈内容填充模板（如React/Vue/PostgreSQL/Supabase等）
- 从Simple Icons获取技术LOGO（保持品牌色）
- 使用Lucide/Heroicons获取功能图标（单色处理）
- 架构图使用SVG绘制（矩形+箭头+文字）
- 党徽使用标准SVG版本（金色单色处理）
- 所有文字为真实技术说明，而非lorem ipsum占位
