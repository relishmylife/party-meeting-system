# 党组织生活会议管理系统 - 启动脚本套件

此项目包含完整的党组织生活会议管理系统，包含前端应用、自动化脚本和详细文档。

## 📁 主要目录

- **`party-meeting-system/`** - React前端应用和项目核心文件
- **`scripts/`** - 系统启动和管理脚本套件
- **`docs/`** - 项目文档和说明资料

## 🚀 快速使用

### 开发环境启动
```bash
cd scripts/
./quick-dev-start.sh
```

### 生产环境部署
```bash
cd scripts/
sudo bash start-all-services.sh
```

## 📚 详细文档

- **[party-meeting-system/README.md](party-meeting-system/README.md)** - 项目完整文档索引
- **[scripts/README.md](scripts/README.md)** - 启动脚本使用指南

## ⚠️ 重要提醒

1. **首次使用**: 请先阅读 `scripts/README.md` 了解脚本使用说明
2. **权限要求**: 生产环境脚本需要sudo权限
3. **环境要求**: Linux系统，Node.js >= 16.0.0
