# 党组织生活会议管理系统 - 配色方案实施指南

## 📋 实施概览

本指南提供将现有科技蓝配色改为内蒙古科技大学红-金-白配色的详细步骤。

---

## 🎯 配色迁移对照表

### 核心颜色替换

| 原配色（科技蓝） | 新配色（红-金-白） | 应用场景 |
|----------------|------------------|---------|
| `#2A64F4` (科技蓝) | `#C8102E` (品牌红) | 主按钮、主要链接、品牌色 |
| `#3A78FF` (浅蓝) | `#A50D25` (深红) | 按钮悬停状态 |
| `#EBF2FF` (淡蓝背景) | `#FEF2F2` (淡红背景) | 图标背景、卡片强调 |
| - | `#F59E0B` (橙金色) | **新增** 核心强调、创建会议按钮 |
| `#333333` (深灰) | `#262626` (标题文字) | 主要标题 |
| `#666666` (中灰) | `#525252` (副标题) | 副标题、次要信息 |
| `#999999` (浅灰) | `#737373` (次要文字) | 描述文字 |
| `#F5F7FA` (页面背景) | `#FAFAFA` (页面背景) | 页面主背景 |
| `#FFFFFF` (白色) | `#FFFFFF` (白色) | 内容区域背景 |

# 将 docs/tailwind-theme-config.js 内容复制到 tailwind.config.js

### 步骤2: 组件样式迁移

#### 2.1 导航栏组件

**文件**: `src/components/Navigation.tsx` (或类似文件)

**原代码示例**:
```jsx
<nav className="bg-white border-b">
  <div className="flex items-center justify-between px-6 py-4">
    <h1 className="text-blue-600 text-xl font-semibold">
      党组织生活会议管理系统
    </h1>
    <button className="bg-blue-600 text-white px-4 py-2 rounded">
      退出登录
    </button>
  </div>
</nav>
```

**新代码**:
```jsx
<nav className="bg-primary-700 border-b border-primary-800">
  <div className="flex items-center justify-between px-6 py-4">
    <h1 className="text-white text-xl font-semibold">
      党组织生活会议管理系统
    </h1>
    <button className="bg-primary-600 hover:bg-primary-700 text-white px-4 py-2 rounded-lg transition-fast">
      退出登录
    </button>
  </div>
</nav>
```

**迁移要点**:
- `bg-white` → `bg-primary-700` (深红背景)
- `text-blue-600` → `text-white` (白色文字)
- `bg-blue-600` → `bg-primary-600` (主按钮红色)
- `hover:bg-blue-700` → `hover:bg-primary-700` (悬停加深)
- `rounded` → `rounded-lg` (圆角8px→12px)

#### 2.2 欢迎横幅组件

**原代码示例**:
```jsx
<div className="bg-blue-600 text-white p-8 rounded-lg mb-6">
  <h2 className="text-2xl font-bold">欢迎，测试管理员</h2>
  <p className="text-blue-100 mt-2">您当前登录为系统管理员，可以访问所有功能模块</p>
</div>
```

**新代码**:
```jsx
<div className="bg-gradient-to-br from-primary-600 to-primary-700 text-white p-8 rounded-lg mb-6 shadow-md">
  <h2 className="text-3xl font-bold">欢迎，测试管理员</h2>
  <p className="text-white text-opacity-90 mt-2">您当前登录为系统管理员，可以访问所有功能模块</p>
</div>
```

**迁移要点**:
- `bg-blue-600` → `bg-gradient-to-br from-primary-600 to-primary-700` (红色渐变)
- `text-blue-100` → `text-white text-opacity-90` (半透明白色)
- `text-2xl` → `text-3xl` (增大标题字号)

**迁移要点**:
- `bg-blue-50` → `bg-primary-50` (淡红图标背景)

- `text-gray-800` → `text-neutral-800` (标题文字)

## ❓ 常见问题

### Q1: 如何保留原配色用于回滚？

**A**: 在 `tailwind.config.js` 中保留原配色：

```js
colors: {
  // 新配色
  primary: { ... },
  
  // 原配色（备份）
  'blue-legacy': {
    500: '#2A64F4',
    600: '#3A78FF',
  }
}
```

### Q2: 部分组件需要保持原配色怎么办？

**A**: 使用条件类名：

```jsx
<button className={isLegacy ? "bg-blue-600" : "bg-primary-600"}>
  按钮
</button>
```

或创建专门的 legacy 样式类。

### Q3: 金色何时使用？

**A**: 仅在以下场景使用金色：
- 创建会议等核心操作按钮
- 重要通知/党建宣传卡片
- 党徽、荣誉标识等政治元素
- 导航高亮状态（文字/下划线）

其他场景使用红色系。
