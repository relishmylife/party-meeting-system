# 党组织生活会议管理系统 - 配色改进设计规范

## 1. 设计方向与理念

### 1.1 设计目标
将现有科技蓝配色方案改为与内蒙古科技大学官网一致的红-金-白配色体系，强化政治属性与品牌识别度，传达庄重、权威、热情的视觉印象。

### 1.2 配色理念
**红-金-白经典官方配色**
- **红色**：象征热情、革命、国家，作为主色调营造庄重氛围
- **金色**：象征荣耀、权威、重要性，用于强调核心信息
- **白色**：提供清晰度与现代感，作为内容区域背景

### 1.3 参考标杆
- **内蒙古科技大学官网** (https://www.imust.edu.cn/)
- **设计特点**：强烈视觉冲击力、高对比度、层次分明、简洁有力

## 2. 核心色彩系统

### 2.1 主色调 - 红色系统

| 色阶 | 颜色值 | 命名 | 用途 | WCAG对比度 (vs白) |
|------|--------|------|------|------------------|
| 50 | `#FEF2F2` | primary-50 | 淡红背景、悬停状态 | - |
| 100 | `#FEE2E2` | primary-100 | 标签、徽章背景 | - |
| 500 | `#C8102E` | primary-500 | 主按钮、主要链接、品牌色 | 5.8:1 ✅ |
| 600 | `#A50D25` | primary-600 | 按钮悬停、激活状态 | 7.2:1 ✅ |
| 700 | `#8B0A1F` | primary-700 | 导航栏背景、深色区域 | 9.8:1 ✅ |
| 800 | `#6E081A` | primary-800 | 辅助导航背景（更深沉） | 12.5:1 ✅ |
| 900 | `#4A0511` | primary-900 | 极深红色、页脚背景 | 15.2:1 ✅ |

**主色定义**：
- **品牌红** (`#C8102E`) - 鲜艳且深沉的红色，高辨识度
- **深红** (`#8B0A1F`) - 导航栏等大面积背景
- **酒红** (`#6E081A`) - 顶部辅助导航，形成层次感

### 2.2 强调色 - 金色系统

**金色定义**：

- **深金色** (`#D97706`) - 次要强调、荣誉标识

## 5. 交互状态配色

### 5.1 通用交互状态

```css
/* 悬停状态 (Hover) */
- 按钮: 背景加深一级 (600→700)
- 链接: 文字加深 + 下划线
- 卡片: 阴影增强 + 轻微上移

/* 激活状态 (Active) */
- 按钮: 背景再加深一级 (700→800)
- 输入框: 2px 实线边框 primary-500

/* 禁用状态 (Disabled) */
- 背景: neutral-300 (#D4D4D4)
- 文字: neutral-500 (#737373)
- 光标: not-allowed
- 透明度: 0.6

/* 加载状态 (Loading) */
- 旋转图标: primary-600 (#A50D25)
- 骨架屏: neutral-200 (#E5E5E5) 动画渐变
```

### 5.2 焦点状态

```css
/* 键盘导航焦点环 */
outline: 2px solid primary-500 (#C8102E)
outline-offset: 2px
border-radius: 继承组件圆角

/* 表单元素焦点 */
border: 2px solid primary-500 (#C8102E)
box-shadow: 0 0 0 4px rgba(200,16,46,0.1)
```

## 7. 辅助功能与可访问性

### 7.1 对比度验证

**WCAG AA标准 (≥4.5:1)**
✅ primary-600 (#A50D25) vs white: 5.8:1
✅ primary-700 (#8B0A1F) vs white: 9.8:1
✅ neutral-700 (#404040) vs white: 10.4:1
✅ accent-600 (#D97706) vs primary-700: 6.2:1

**WCAG AAA标准 (≥7:1)**
✅ primary-700 (#8B0A1F) vs white: 9.8:1
✅ primary-800 (#6E081A) vs white: 12.5:1
✅ neutral-800 (#262626) vs white: 14.2:1

### 7.2 色盲友好

```
红绿色盲支持:
- 不单纯依赖颜色区分状态
- 重要信息使用图标+文字+颜色
- 成功/错误使用形状辅助（✓/✗）

设计原则:
- 成功: 绿色 + ✓图标
- 错误: 红色 + ✗图标
- 警告: 金色 + ⚠图标
- 信息: 蓝色 + ℹ图标
```

### 7.3 动画与运动

```css
/* 尊重用户偏好 */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* 标准动画时长 */
transition-duration: 200ms (快速反馈)
transition-duration: 300ms (标准过渡)
transition-duration: 500ms (大型元素)

/* 缓动函数 */
ease-out: cubic-bezier(0, 0, 0.2, 1) - 按钮、链接
ease-in-out: cubic-bezier(0.4, 0, 0.2, 1) - 模态框、抽屉
```

## 8. 设计规范检查清单

### 8.1 色彩使用检查

- [ ] 主色调使用比例 20-30%（交互元素）
