# 党组织生活会议管理系统 - 启动脚本套件

## 📁 目录说明

此文件夹包含完整的系统启动和部署脚本套件，专门为党组织生活会议管理系统设计。所有脚本都经过精心优化，支持一键操作、健康检查、日志记录等企业级功能。

## 🚀 快速开始

### 开发环境
```bash
cd scripts/
# 快速启动开发服务器（无需sudo）
./quick-dev-start.sh
```

### 生产环境
```bash
cd scripts/
# 完整系统启动（需要sudo）
sudo bash start-all-services.sh
```

### 停止服务
```bash
cd scripts/
# 停止所有服务（需要sudo）
sudo bash stop-all-services.sh --quick
```

## ⚡ 使用场景

### 🛠️ 开发场景
- **日常开发**: 使用 `quick-dev-start.sh`
- **功能测试**: 快速启动Vite开发服务器
- **热重载**: 支持实时代码更新

### 🏢 生产场景
- **系统部署**: 使用 `start-all-services.sh`
- **服务管理**: 使用 `stop-all-services.sh` 进行服务控制
- **状态监控**: 检查系统运行状态

### 📚 学习场景
- **操作演示**: 运行 `脚本使用演示.sh`
- **文档学习**: 阅读相关说明文档

## 🔧 依赖要求

- **操作系统**: Linux (Ubuntu 20.04+ / CentOS 7+)
- **Node.js**: >= 16.0.0
- **pnpm**: 最新版本
- **Nginx**: >= 1.18
- **权限**: 部分脚本需要sudo权限
