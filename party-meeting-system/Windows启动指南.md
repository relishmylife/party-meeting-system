## ⚙️ 环境要求

### 系统要求
- **操作系统**: Windows 10/11 (64位)
- **内存**: 最低2GB，推荐4GB+
- **磁盘**: 最低1GB可用空间

### 软件要求
1. **Node.js** (必需)
   - 下载地址: https://nodejs.org/
   - 推荐版本: v18+ 或 v20+
   - 安装时会自动安装 npm

2. **pnpm** (推荐)
   ```bash
   # 安装 pnpm (可选但推荐)
   npm install -g pnpm
   ```
   - 优点: 更快、更节省磁盘空间
   - 如果未安装，脚本会自动使用 npm

## 🔧 常见问题解决

### ❌ 问题1: "Node.js 未安装"
**解决方案**:
1. 访问 https://nodejs.org/
2. 下载并安装 LTS 版本
3. 重启命令行窗口
4. 重新运行启动脚本

### ❌ 问题2: "端口被占用"
**解决方案**:
脚本会自动检测并询问是否终止占用进程，选择 `y` 即可。

**手动解决**:
```bash
# 查看占用端口的进程
netstat -ano | findstr "5173"
taskkill /PID <进程ID> /F
```

### ❌ 问题3: "依赖安装失败"
**解决方案**:
1. 检查网络连接
2. 清理缓存重新安装:
   ```bash
   # 删除缓存目录
   rmdir /s /q node_modules
   del package-lock.json
   # 重新运行启动脚本
   ```

### ❌ 问题4: "权限不足"
**解决方案**:
- Windows用户账户控制(UAC)可能会阻止某些操作
- 右键点击脚本 → "以管理员身份运行"

### ❌ 问题5: "访问地址打不开"
**解决方案**:
1. 确认服务已启动 (命令行窗口显示"Local:"信息)
2. 检查防火墙设置
3. 尝试访问 http://127.0.0.1:5173
4. 重启浏览器
